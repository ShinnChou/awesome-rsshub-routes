## üìñ How to Use

### Using RSSHub Routes

1. **Choose an RSSHub instance**
   - Official: `https://rsshub.app`
   - Self-hosted: See [deployment docs](https://docs.rsshub.app/deploy/)

2. **Build the feed URL**
   ```
   Instance URL + Route = Feed URL

   https://rsshub.app + /weibo/search/hot = https://rsshub.app/weibo/search/hot
   ```

3. **Route parameters**

   Replace `:param` with actual values:
   ```
   /bilibili/user/video/:uid
                        ‚Üì
   /bilibili/user/video/546195  ‚Üê Replace with actual UID
   ```

### Using Official RSS Feeds

Simply copy the feed URL into your RSS reader ‚Äî no assembly required.
